## Installation

```bash
# installing prisma
$ pnpx prisma

$ pnpm install

# generating client
$ prisma generate
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Accessing SwaggerUI

```bash
localhost:3000/api
```
